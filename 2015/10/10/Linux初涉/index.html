Linux初涉

发表于 2015-10-10   |   分类于 Linux   |  

说的通俗点，那是为了别人看你的眼光透着羡慕。我想每一个程序员都想像电影里的黑客一样，在命令行中噼里啪啦敲一大丢你看不懂的命令，然后，稀里哗啦的显示一大丢你貌似看得懂的东西。说的文艺点，那是为了梦想，为了去圆当初吹过的牛逼，就要花一些别人睡觉的日日夜夜去苦逼的”望闻问切”,有时候，目的也许只是为了得到一个肯定，证明自己也是可以做到别人能做到的事，达到别人曾达到的高度。

当初从瘟都死机子，到Mac机，然后初涉Linux，最后因为工作原因，又回到瘟都死，操作系统孰强孰弱，每个人都有各自的看法。作为Mac和Chrome的狂热爱好者，就不提鄙陋之见了。好了，叨叨了一圈，直奔主题吧。

环境：Mac Terminal + Linux CentOS + Mou

##tar比较重要

tar [主选项+辅选项] 文件或目录

-v 详细报告tar处理的文件信息。如无此选项，tar不报告文件信息。

-c 创建新的档案文件。如果用户想备份一个目录或是一些文件，就要选择这个选项。

-f 使用档案文件或设备，这个选项通常是必选的。

-z 用gzip来压缩/解压缩文件，加上该选项后可以将档案文件进行压缩，但还原时也一定要使用该选项进行解压缩。

解包：tar xvf FileName.tar

打包：tar cvf FileName.tar DirName

打包后的后缀是.tar

解压：tar zxvf FileName.tar.gz

压缩：tar zcvf FileName.tar.gz DirName

打包后的后缀是.tar.gz

gzip 选项 [文件]

压缩文件，压缩后的文件格式是.gz，有两个特点

1.只能压缩文件，不能压缩目录

2.不保留原文件

gunzip 或者gzip -d 解压缩

file xxx 可查看xxx是什么文件类型

.zip

Zip 选项[-r] [压缩后文件]

压缩文件 zip service.zip /etcrvices

压缩目录 zip –r test.zip /test

解压缩 unzip 压缩文件或目录名字

bzip2 选项[-k] [文件名称]

压缩文件，压缩后的后缀是.bz2

解压缩

bunzip2 选项[-k] [压缩文件]

-k 是否保留原文件

进程管理

ps

-a 显示所有用户的进程

-u 显示用户名和启动时间

-x 显示没有控制终端的进程

-l 长格式显示

-w 宽行显示，可以使用多个w进行加宽显示

ps –le 或者 ps –aux 查看所有用户执行的进程的详细信息

ps –le – sort pid 排序

pstree | more 查看进程树，父子进程

kill 进程号

killall 结束所有进程

kill -1 进程号 （重启） ==restart

pgrep httpd

kill -1 pgrep httpd 重启

pkill httpd 关闭
